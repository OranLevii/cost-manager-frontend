# Cost Manager Frontend

Welcome! This is a cost management application built with React and Vite. Follow the simple steps below to get started.

## Getting Started

### Step 1: Install Dependencies
First, make sure you have all the required packages installed. Open your terminal in the project folder and run:

```bash
npm install
```

This will install all the necessary dependencies for the project.

### Step 2: Start the Development Server
Once the installation is complete, start the development server by running:

```bash
npm run dev
```

### Step 3: Open the Application
After running the command, you should see a message in your terminal showing the local development URL (usually `http://localhost:5173`). 

Simply:
- Copy the URL from your terminal
- Open your web browser
- Paste the URL in the address bar and press Enter

 Your cost manager application should now be running!

##  First-Time Setup - Important!

**Before you start using the application, you need to configure the exchange rates JSON file:**

1. Once the application is open in your browser, navigate to the **Settings** page
2. In the settings, you'll find a field to enter the JSON file URL
3. Enter the following URL:
   ```
   https://oranlevii.github.io/cost-manager-rates/rates.json
   ```
4. Save your settings

This will allow the application to fetch the latest exchange rates for currency conversion.

##  Available Scripts

This cost manager application allows you to:
- Add and track your expenses
- View reports and analytics
- See visual charts (bar charts and pie charts)
- Manage settings including exchange rates
